Welcome to Gink!
================

Gink is a versioned, eventually consistent database system with a protocol-first
sync model. Start with :doc:`getting_started` for Python usage and
:doc:`examples` for container examples.

For implementation and protocol context, also see the project-level Markdown
docs in the repository root, especially ``docs/architecture.md``,
``docs/data_model.md``, ``docs/consistency.md``, ``docs/syncing.md``, and
``docs/security.md``.
